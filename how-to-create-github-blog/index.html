如何搭建一个免费的 GitHub 博客

是的，你没看错，这篇文章就是教你如何在 GitHub 上免费搭建一个自己的博客。全程不需要花一分钱，只需要一个 GitHub 账号。

什么是 GitHub Pages？

GitHub Pages 是 GitHub 提供的免费静态网站托管服务。你可以把自己的博客、项目文档或者个人主页托管在 GitHub 上，然后通过 username.github.io 这样的域名访问。

搭建步骤

1. 创建一个仓库

登录你的 GitHub 账号，点击右上角的 “+” → “New repository”。

仓库名设置为：你的用户名.github.io

比如我的就是：1336857019-sketch.github.io

4. 访问你的博客

等待几分钟后，打开浏览器访问：https://你的用户名.github.io

恭喜你！博客已经上线了！🎉
