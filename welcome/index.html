欢迎来到 Nick’s Blog 🎉

你好！我是 Nick，一名电商设计师，来自上海。

很高兴你找到了我的小天地！

关于这个博客

这个博客是用 Hexo + GitHub Pages 搭建的。

作为一个设计狮，平时大部分时间都在和电商页面、UI设计打交道。

建这个博客呢，主要是想：

  • 📝 记录一些工作和生活中的思考
  • 🌈 分享有趣的发现
  • 💡 记录学习的点滴

关于我

  • 🖥️ 职业：电商设计师
  • 📍 地点：上海
  • 🎯 爱好：折腾各种工具、刷新闻、看看科技前沿

博客技术

这个博客使用：

  • Hexo - 静态博客框架
  • GitHub Pages - 免费托管
  • Landscape 主题 - 简约大方

如果你也想定制自己的博客，欢迎交流！🤝

感谢你的来访～

有问题可以给我留言，或者直接来聊天！

Nick 🫛
